# Property Management Portal - Color Palette
## Airbnb-Style with Warm Wooden Tones

This document outlines the comprehensive color palette configured for the property management portal, combining Airbnb-inspired red branding with warm wooden/earthy tones from luxury property advertisements.

## Overview

The color scheme emphasizes:
- **Airbnb-style red** (`#FF385C`) as the primary brand color
- **Warm wooden/earthy tones** (`#805A3A`, `#E8A06A`) for secondary elements
- **Rich, warm colors** throughout inspired by property ads
- **Full dark mode support** with carefully crafted variants
- **All 10 Mishika Chelekom colors** configured and ready to use

## Color Configuration

Colors are configured in `priv/mishka_chelekom/config.exs` using the `css_overrides` option with `:merge` strategy, which means:
- Custom colors override defaults
- Unspecified colors keep default values
- Changes require CSS rebuild to take effect

After color changes, rebuild assets:

## ‚ôø Accessibility & WCAG Compliance

### WCAG 2.1 AA Standards
All colors in this palette are tested against WCAG 2.1 AA standards:
- **Normal Text** (body, <24px): Minimum **4.5:1** contrast ratio ‚úÖ
- **Large Text** (‚â•24px regular or ‚â•18.5px bold): Minimum **3:1** contrast ratio ‚úÖ
- **UI Components** (buttons, icons, borders): Minimum **3:1** contrast ratio ‚úÖ

### Contrast-Tested Combinations

#### ‚úÖ Passing Combinations
- **Primary text (`#332A20`)** on light backgrounds - Passes WCAG AAA (13-15:1)
- **Off-white text (`#FAF9F7`)** on dark backgrounds - Passes WCAG AAA (10-15:1)
- **White text** on most colored buttons - Passes WCAG AA
- **Colored text** on white backgrounds - Passes WCAG AA

#### ‚ö†Ô∏è Special Considerations

**Silver Text (`#757575`)**
- ‚ö†Ô∏è **Does NOT meet AA** on warm beige backgrounds (`#F8F5F0`, `#F5F1ED`)
- ‚úÖ **Solution**: Use `#4A4A4A` for primary text instead
- ‚úÖ **Reserve `#757575`** for secondary/tertiary text only

**Warning Buttons (`#D4A017`)**
- ‚ö†Ô∏è **White text fails AA** on warning button background
- ‚úÖ **Solution**: Use dark text (`#332A20`) or darker background (`#B8860B`) with white text

**Dawn/Terracotta Buttons (`#E8A06A`)**
- ‚ö†Ô∏è **White text fails AA** on terracotta background
- ‚úÖ **Solution**: Use dark text (`#332A20` or `#6B4229`) instead of white

### Color Differentiation

**Primary vs Danger Red**
- **Primary**: `#FF385C` - Used for CTAs, brand
- **Danger**: `#B71C1C` - Used for errors, destructive actions (refined from `#C62828` for better differentiation)

The danger color has been refined to `#B71C1C` to clearly distinguish error states from primary brand actions.

### Dark Mode Accessibility

All dark mode colors are carefully tuned for optimal readability:
- **Primary text**: `#FAF9F7` (off-white) instead of pure white - reduces glare
- **Backgrounds**: Warm dark grays (`#1A1815`, `#2A2724`) instead of pure black
- **Colored text**: Brighter variants for better contrast on dark backgrounds

### Button Text Guidelines

**Important**: Some buttons require dark text instead of white for WCAG AA compliance.
